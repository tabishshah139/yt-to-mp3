YouTube to MP3

Paste a YouTube link and download the audio

Browser Extension (Recommended)
YouTube blocks server-side downloads from datacenter IPs. Our Chrome extension extracts audio directly from your browser — works for any video.

Install:
1. Download extension files (or copy the extension/ folder from the project)
2. Open chrome://extensions in Chrome
3. Enable Developer mode (top right toggle)
4. Click "Load unpacked" and select the extension folder
5. Go to any YouTube video — a red "Download MP3" button will appear below the video
